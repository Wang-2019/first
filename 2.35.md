## i是int类型常量
## j是int类型变量
## k是常量引用，绑定到常量i上
## p是int类型指针，指向常量i
## j2是int类型常量
## k2是常量引用，绑定到常量i上
```C++
#include<iostream>

int main(){
	const int i = 42;
	auto j = i;
	const auto &k = i;
	auto *p = &i;
	const auto j2 = i,&k2 = i;
	cout <<"i的类型是："<<typeid(i).name()<<endl;
	cout <<"j的类型是："<<typeid(j).name()<<endl;
	cout <<"k的类型是："<<typeid(k).name()<<endl;
	cout <<"p的类型是："<<typeid(p).name()<<endl;
	cout <<"j2的类型是："<<typeid(j2).name()<<endl;
	cout <<"k2的类型是："<<typeid(p2).name()<<endl;
	return 0;
}
```
